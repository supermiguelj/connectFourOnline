Required Installs
MySQL
npm
nodejs
react
Maven

Steps
  Run npm install on your terminal (possibly as an administrator if necessary) in the directory where app.js is.
  Create a database in mysql command client called "Connect4DB".
  Get tables.sql and user_stats.sql off the github (connectFourOnline/server/connectfour/src/main/resources/SQL)
  Copy all of the SQL commands from tables.sql and user_stats.sql to mysql.
  Run ConnectfourApplication.java (connectFourOnline/server/connectfour/src/main/java/com/connectfour).
  In the terminal where the directory for app.js is, type in "npm start"
  Access the website using "http://localhost:3000"
